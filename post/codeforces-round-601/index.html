B. Send Boxes to Alice (Easy/Hard Version) (1s 256Mb)

题目大意

nnn个排列好的盒子，每个盒子里有aia_iai​块巧克力。可以花费1秒的时间将一块巧克力从一个盒子移动到左右相邻的盒子，问最少可以用多少秒，使得每个盒子里的巧克力数都可以被kkk整除。其中kkk是大于的任意数。

n≤105(E)/106(H)n \leq 10^5(E)/10^6(H)n≤105(E)/106(H)
ai≤1(E)/106(H)a_i \leq 1(E)/10^6(H)ai​≤1(E)/106(H)

分析

Easy：每个盒子要么有一块要么没有。对于specifickkk，将相邻的kkk个1整合所花费时间最少。可以暴力枚举所有sumsumsum（巧克力总数）的因数为kkk求解，取答案最小。小于10510^5105的数最多有128个因子。sum==1sum == 1sum==1时输出-1。O(128∗n)O(128*n)O(128∗n)。

Hard：
​​
使用前缀和，移动等价于某个sis_isi​加一或者减一。既然最后每个aia_iai​可以被kkk整除（不同于Easy，可以是k的倍数），显然最终情况等价于每个sis_isi​也可以被kkk整除。

对于每一个sis_isi​，使用min(si%k,k−si%k)min(s_i\%k, k - s_i\%k)min(si​%k,k−si​%k)秒（贪心）使合法，由于si≤si+1s_i \leq s_{i+1}si​≤si+1​，所以不会出现逆序的非法情况。可以O(n)O(n)O(n)求解一个kkk的答案。

观察可知答案对应的kkk一定是素数（合数kkk的情况被其素因子包括），对于一个小于101210^{12}1012的数字，最多有12个不同的素因子，唯一分解即可。

代码

const int maxn = 1e6+10;
int n;
ll a[maxn], s[maxn];

ll work(ll k)
{
	ll res = 0;
	for(int i = 1; i < n; i++) res += min(s[i]%k, k-s[i]%k);
	return res;
}

int main()
{
	n = read();
	for(int i = 1; i <= n; i++) a[i] = read(), s[i] = s[i-1]+a[i];
	if(s[n] == 1)
	{
		printf("-1\n"); return 0;
	}
	ll ans = 9e18, tmp = 2, num = s[n];
	while(tmp * tmp <= num)
	{
		if(num%tmp == 0)
		{
			ans = min(ans, work(tmp));
			while(num%tmp == 0) num /= tmp;
		}
		tmp++;
	}
	if(num > 1) ans = min(ans, work(num));
	printf("%I64d\n", ans);
	return 0;
}
