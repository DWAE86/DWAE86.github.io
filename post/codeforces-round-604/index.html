E1. Beautiful Mirrors (2s 256Mb)

题目大意

nnn个排列好的镜子，每面镜子有一个概率pip_ipi​，表示你问他你好不好看，他说你好看的概率。每一天，你问一面镜子iii。成功了，明天你问下一面i+1i+1i+1，不成功明天问第1面。第一天你问第1面，求冲过最后一面时用的天数的期望值。

n≤2⋅105n \leq 2\cdot 10^5n≤2⋅105

分析

疯狂自闭概率期望。

设eie_iei​为在第i面镜子前，你需要几天冲完。边界en+1=0e_{n+1}=0en+1​=0 。由定义

即询问的一天，以及两种询问结果的情况乘以概率。根据推导

代码

int main()
{
	int n = read();
	for(int i = 1; i <= n; i++) p[i] = 1LL*read()*inv(100)%mod;
	int ans = 1, tmp = 1;
	for(int i = 1; i < n; i++)
	{
		tmp = 1LL*tmp*p[i]%mod;
		ans = (ans+tmp)%mod;
	}
	ans = 1LL*ans*inv((1LL*tmp*p[n])%mod)%mod;
	printf("%d\n", ans);
	return 0;
}
